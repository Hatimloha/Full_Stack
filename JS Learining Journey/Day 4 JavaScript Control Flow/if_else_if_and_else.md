# Theory:
> Control flow allows your program to make decisions and execute code based on conditions.

### Basic if Statement:
```js
if (condition) {
  // code to run if condition is true
}
```

### if-else Statement:
```js
if (condition) {
  // code if true
} else {
  // code if false
}
```

###  if-else if-else Chain:
```js
if (condition1) {
  // code if condition1 is true
} else if (condition2) {
  // code if condition2 is true
} else {
  // code if none are true
}
```
