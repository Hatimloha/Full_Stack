# Theory:
> An array is a special variable that can hold multiple values in a single variable.

### Syntax
```js
let fruits = ["apple", "banana", "mango"];
```
- Each value in an array has an index starting from 0:
    - fruits[0] → "apple"
    - fruits[1] → "banana"

### Array Properties and Methods:
Property: Method Description
- length: Returns number of items
- push(): Adds element to the end
- pop(): Removes last element
- unshift(): Adds to the beginning
- shift(): Removes first element
- indexOf(): Finds the index of a value
- includes(): Checks if a value exists
- join(): Combines elements to a string
- sort(): Sorts the array (alphabetically by default)
