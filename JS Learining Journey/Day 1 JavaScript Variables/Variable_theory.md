# Variables 

## There are three type of variables:
- var (function-scoped, outdated)
- let (function-scoped, prefered)
- const (function-scoped, for constants)

### Syntax:
```js
let name = "hatim";
var age = 25;
const PI = 3.14;
```

### Rules
- Variable name are case-sensitive
- Cannot Start with a number
- Can contain letter, digit, special character's
- Avoid using reserved keyword
- Variable name cannot start from the number, () etc.
